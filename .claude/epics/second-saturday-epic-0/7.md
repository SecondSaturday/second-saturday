# Task: Set Up Capacitor Mobile

## Description
Initialize Capacitor to wrap the Next.js app for iOS and Android. Configure native projects, install essential plugins, and test builds on simulators.

## Acceptance Criteria
- [ ] @capacitor/core and @capacitor/cli installed
- [ ] Capacitor project initialized (capacitor.config.ts)
- [ ] Next.js build output configured for Capacitor
- [ ] iOS platform added
- [ ] Android platform added
- [ ] Essential Capacitor plugins installed (Camera, Filesystem, PushNotifications)
- [ ] App name, bundle ID, and version configured
- [ ] Xcode project set up and builds on simulator
- [ ] Android Studio project set up and builds on emulator
- [ ] Apple Developer credentials configured in Xcode

## Technical Details
- Run `npx cap init` to initialize
- Configure next.config.js for static export (output: 'export')
- Run `npx cap add ios` and `npx cap add android`
- Install: @capacitor/camera, @capacitor/filesystem, @capacitor/push-notifications
- Configure capacitor.config.ts with app ID (com.secondsaturday.app)
- Set up signing certificates in Xcode
- DAY 1 CRITICAL: Apple Developer Program enrollment (24-48hr approval)

## Dependencies

- [ ] Task 002 completed (UI framework for proper rendering)
- [ ] Apple Developer account approved
- [ ] macOS with Xcode installed
- [ ] Android Studio installed
