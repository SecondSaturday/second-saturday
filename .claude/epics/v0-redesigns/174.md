# Task: Newsletter header redesign with serif font

## Description

Redesign the newsletter header to use `ProfileHeaderImageLayout` (display mode) with actual circle images, add a serif font for the circle name, and simplify the top bar. Fix PL4 (null image URLs) by passing actual `circle.iconUrl`/`circle.coverUrl` from queries.

## Acceptance Criteria

- [ ] Newsletter header shows full-width cover image banner with circle icon overlapping bottom edge
- [ ] Circle name rendered in large serif font, centered below icon
- [ ] Slug/tagline in smaller muted text below name
- [ ] Actual `circle.iconUrl` and `circle.coverUrl` passed from queries (PL4 fix)
- [ ] Top header bar simplified to just back arrow (or overlaid on cover)
- [ ] Serif font added to project (e.g., Playfair Display or Lora via next/font)
- [ ] Prompt section headings also use serif font

## Technical Details

### PL4 fix: pass actual image URLs
- `src/app/dashboard/page.tsx` — the `DesktopCircleNewsletter` component currently passes `iconUrl: null, coverUrl: null`. Update to pass `circle.iconUrl` and `circle.coverUrl` from the `getCircle` query (which returns storage URLs).
- `src/app/dashboard/circles/[circleId]/page.tsx` — same fix in the `circleInfo` object.

### Newsletter header redesign
- `src/components/newsletter/NewsletterView.tsx` (89 lines) — replace the current small icon + text header with:
  1. `ProfileHeaderImageLayout` in display mode (`editable={false}`)
  2. Centered serif circle name below
  3. Muted tagline text

### Serif font
- Add serif font in `src/app/layout.tsx` via `next/font/google` (e.g., `Playfair_Display`)
- Export CSS variable (e.g., `--font-serif`)
- Apply via Tailwind class `font-serif` in `tailwind.config.ts`
- Use in NewsletterView header and PromptSection headings

### Simplify top bar (FR3)
- Newsletter page wrappers: remove or minimize the `<header>` that duplicates circle name

### Files affected
